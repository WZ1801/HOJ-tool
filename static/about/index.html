What HOJ Tool?

关于 HOJ Tool

HOJ Tool 是一款专为 HOJ 在线评测系统设计的智能辅助工具，旨在帮助用户更高效地学习和使用 HOJ 平台。

当前版本：v5.0.0

基于 Web 技术构建，提供友好的用户界面和强大的功能。

自动刷题

利用 AI 技术智能解答编程题目，提高学习效率

数据统计

详细的提交统计和排行榜信息，掌握平台状况，分清局势

账号管理

便捷的账号封禁功能，正义降临

简要使用帮助

  • 配置信息设置

    在设置页面中配置您的 OJ 平台地址、API 地址、用户名密码，以及 AI 服务地址和浏览器驱动路径。

  • 自动刷题功能

    进入自动刷题页面，选择模式，系统将自动解答并提交答案。

  • 数据统计查看

    在统计页面查看所有提交记录、通过率、排行榜等详细数据。

  • 账号封禁功能

    使用账号封禁功能可以快速处理违规账号，维护平台秩序。
