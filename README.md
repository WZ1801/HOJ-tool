# OJ自动刷题爬虫

#### 介绍
利用AI（360bot）对HOJ类OJ的自动爬虫刷题系统，适配于Windows

#### 安装教程

1.  克隆仓库至本地
2.  安装谷歌浏览器和对应的爬虫驱动
3.  安装Python第三方库（运行install_packages.bat）
4.  修改main.py，手动配置爬虫驱动、OJ地址、360bot_Cookies

#### 使用说明

1.  打开main.py后等待登录
2.  输入训练号，开始刷题

⚠️鼠标和键盘运行时不能使用（鼠标要滚动页面），输入法调为英文（会输入路径）
